# geosampa-e2e
Testes "end-to-end" do portal geosampa

## Setup
```bash
# clone o repositório
git clone https://github.com/geoinfo-smdu/geosampa-e2e.git

# instale as dependências
cd geosampa-e2e
npm i

# inicie em modo client
npm run test:run

# inicie em modo de desenvolvimento (com browser e cypress)
npm run test:open
```
